# SunSpot: The Ephemeral City

Redefining Navigation through Human Light Preferences

## About

SunSpot is an interactive web application that provides intelligent routing based on solar exposure and shadow patterns in urban environments. The application helps users find optimal paths that maximize or minimize sunlight exposure based on their preferences.

## Features

- **Dual-Mode Routing**: Choose between sun-seeking and shade-preferring routes
- **Real-time Shadow Visualization**: Interactive 3D map showing building shadows
- **Time-based Scenarios**: View how shadows change throughout the day
- **Modern UI**: Beautiful glass-morphism design inspired by VisionOS

## GitHub Pages Setup

This project is configured to be hosted on GitHub Pages. The main file is `index.html` in the root directory.

### How to Enable GitHub Pages:

1. Go to your repository: https://github.com/MinweiZhao/ugod5104-sunspot
2. Click on **Settings** tab
3. Scroll down to **Pages** section (in the left sidebar)
4. Under **Source**, select:
   - **Branch**: `main`
   - **Folder**: `/ (root)`
5. Click **Save**
6. Your site will be available at: `https://minweizhao.github.io/ugod5104-sunspot/`

## Local Development

Simply open `index.html` in a web browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (http-server)
npx http-server
```

Then visit `http://localhost:8000`
